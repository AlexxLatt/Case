# MegaAtom — WordPress тема и модуль поиска

Краткое описание
-----------------
Проект — кастомная тема WordPress `gorn-2` и набор модулей поиска/корзины, разработанные для сайта megaatom.ru. Тема содержит интегрированный фронтенд-поиск по артикулам (включая загрузку файлов Excel/CSV), защищённую логику корзины на куках, обработку заказов и страницу управления заказами в админке.

Ключевые возможности
--------------------
- **Поиск товаров:** мультирежимный поиск (по одному артикулу, списком, загрузка файла) с AJAX и генерацией таблицы результатов.
- **Корзина:** клиентская корзина в куках с шифрованием/сжатием, валидацией UID/подписей и AJAX-эндпоинтами для добавления/удаления.
- **Обработка заказов:** форма отправки заказа и серверная обработка с сохранением в таблицу `wp_custom_orders` и админ-страницей для просмотра заказов.
- **Админ-страницы и настройки:** страница «Заказы», настройки валют и логики наценок, генератор таблиц и шорткод для встраивания функционала.
- **Фронтенд:** `assets/js/megaatom-main.js` — основной скрипт для главной страницы: поиск, модальные окна, drag&drop загрузка, отправка форм.

Технологии
---------
- WordPress (PHP)
- PHP 8.x+ (модули: OpenSSL, zlib)
- JavaScript (vanilla + AJAX)
- HTML/CSS

Структура проекта (важные файлы)
--------------------------------
- `wp-content/themes/gorn-2/functions.php` — загрузка модулей темы и регистрация скриптов/шорткодов.
- `wp-content/themes/gorn-2/assets/js/megaatom-main.js` — фронтенд логика поиска и форм.
- `wp-content/themes/gorn-2/inc/megaatom-search/admin-orders-page.php` — админ-интерфейс «Заказы».
- `wp-content/themes/gorn-2/inc/megaatom-search/cart-handler.php` — безопасная логика корзины и AJAX-обработчики.
- `wp-content/themes/gorn-2/inc/megaatom-search/shortcode.php` — шорткод для встраивания поиска/форм.
- `wp-config.php` — здесь можно задать константы для шифрования/подписей (рекомендуется настроить `COOKIE_ENCRYPTION_KEY` и `CART_SIGNATURE_KEY`).
