## 📋 Краткое описание

Проект — кастомная тема WordPress `gorn-2` и набор модулей поиска/корзины, разработанные для сайта [megaatom.ru](https://megaatom.com). 

**В репозитории представлены:**
- ✅ **v1.0 (Production):** Базовая реализация, работающая на боевом сайте.
- ✅ **v2.0 (R&D):** Архитектурный рефакторинг с улучшениями безопасности и производительности (демонстрация в портфолио).

## 🚀 Ключевые возможности

### Поиск товаров
- Мультирежимный поиск: по одному артикулу, списком, загрузка файла (Excel/CSV)
- AJAX-запросы с генерацией таблицы результатов
- **v2.0:** Параллельные запросы через `curl_multi` (ускорение до 50×)

### Корзина
- Клиентская корзина в куках с шифрованием/сжатием
- Валидация UID и подписей для защиты от манипуляций
- AJAX-эндпоинты для добавления/удаления товаров
- **v2.0:** Шифрование AES-256-CBC + сжатие gzip + флаги HttpOnly/Secure/SameSite

### Обработка заказов
- Форма отправки заказа и серверная обработка
- Сохранение в таблицу `wp_custom_orders`
- Админ-страница для просмотра и экспорта заказов

### Админ-страницы и настройки
- Страница «Заказы» в админке
- Настройки валют, логики наценок, генератор таблиц
- Шорткод для встраивания функционала на любые страницы

### Фронтенд
- `assets/js/megaatom-main.js` — основной скрипт: поиск, модальные окна, drag&drop, валидация форм
